# Nandi

![nandi](https://user-images.githubusercontent.com/2285130/56881872-bef6f500-6a59-11e9-8936-04d3861b6dce.gif)

Friendly migrations for people who don't want to take down their database
to add a column!

## Generate documentation

```sh
bundle exec yard
```

## Run tests

```sh
bundle exec rspec
```
